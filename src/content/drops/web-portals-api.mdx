🚪 **A Web Portals API** é um [padrão](https://wicg.github.io/portals/) que busca otimizar a aplicação que damos atualmente aos `<iframe />`.

🕵️‍♂️ **Web Portals** são criados com a tag `<portal src="..." />`.

⤬ **Web Portals** estão sempre no contexto *top-level* do browser, possuem navegabilidade independente e somente podem ser acessados via `postMessage`.

\[demo]

👆 Como visto acima, dentre as principais vantangens, **Web Portals permitem é carregamento prévio de conteúdo e transição animada na navegação entre páginas.**

⚡️ Além disso, **Web Portals permitir**á combinar os beneficios da criação de **Multi-Page Applications** com arquitetura de **Single-Page Applications**.
