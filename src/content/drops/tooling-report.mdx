👉 Você conhece o <A href="https://tooling.report/">tooling.report</A>?

![Tooling.Report](https://user-images.githubusercontent.com/1680157/87510877-bf71b080-c64a-11ea-9f20-67073c177345.png)

Tooling.Report é um serviço criado pelo times do <A href="https://web.dev/">web.dev</A> e <A href="https://github.com/GoogleChromeLabs">GoogleChromeLabs</A> que visa nos ajudar escolher a melhor ferramenta de build de acordo com o problema que queremos resolver.

O serviço também promvove guidelines de boas práticas de acordo com cada ferramenta.

Atualmente é oferecido guidelines para:

🧙‍♀️ <A href="http://browserify.org/">Browserify</A>

📦 <A href="https://parceljs.org/">Parcel</A>

🆁 <A href="https://rollupjs.org/">Rollup</A>

⬢ <A href="https://webpack.js.org/">Webpack</A>
