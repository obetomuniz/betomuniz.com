🔑 A [Web OTP API](https://web.dev/web-otp/) visa otimizar a experiência durante o fluxo de autenticaçao via SMS na web.

💡 E para isso, sua proposta é reduzir a fricção desde a requisição de códigos de autenticação até a identificação e aplicação automática nas nossas aplicação web.

👉 Se seu browser já suporta essa [Web API](http://wicg.github.io/WebOTP), é possível testar [aqui](https://web-otp-demo.glitch.me/).
