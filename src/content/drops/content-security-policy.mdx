💂‍♀️ **Content Security Policy (CSP)** é um header HTTP que permite detalhar de onde pode ser aplicado recursos (estilos, scripts, etc) de um site

🕷 Esse recurso é eficiente contra ataques **cross-site script (XSS)**

👏 Adote CSP e evite as diretivas **unsafe-inline** ou **unsafe-eval**
