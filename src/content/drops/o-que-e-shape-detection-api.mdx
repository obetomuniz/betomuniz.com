**🚪 ShapeDetection API** são especificações web que oferecem funcionalidades para trabalharmos **detecção de elementos em imagem**. E **nesse ”guarda-chuva” temos as seguintes Web APIs**:

⭐️ **Barcode API**

👉 API voltada para implementação de leitores e geradores de código de barras e QR Code diretamente da Web.

⚡️ **Face Detection API**

👉 API que permite ao Browser de forma nativa a ativação de reconhecimento facial. Aplicações com esse tipo de recursos são infinitos.

🌟 **Text Detection API**

👉 API que identifica textos em imagens. Já pensou a aplicação disso em tradução? Poderemos ir para o Japão sem muitos problemas hahaha

😉 **Nas próximas semanas irei detalhar cada uma dessas APIs.**
